# Snake
Snake game in python.

Written just for fun (and to get familiar with Pygame, PySimpleGUI and git).

## Technologies
python 3

pygame

pySimpleGUI

## Setup

`./snake`

(If access denied, try `chmod u+x snake`)

(You need to have [python] and pip installed)
